Idiomatic Rust in Simple Steps

Idiomatic Rust in Simple Steps (IRISS) is a step-by-step guide to learning Rust no programming knowledge required. By focusing on idioms, the series aims to get you productive as quickly as possible.

There are two parts to IRISS, a YouTube Series that's easy to watch, and a written guide that provides more details.
